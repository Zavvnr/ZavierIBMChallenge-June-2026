# Extra Module

## Mainly for the third Agent that Responds to the Questions

**HIGH IMPORTANCE!**

Lead and Analyst Agent should be Muted (not stopped) When a user ask question to the Agent. In other words, they should still generate commentary but their audio is muted until the third agent finish its sentences.

## The Third Agent

The third agent serves as the third commentator that only respond when a user ask a question. For example:
Q: How did the latest chance that led to the goal first happened?
A: Messi Drags the defender which opens the space, Di Maria runs through the wing.
